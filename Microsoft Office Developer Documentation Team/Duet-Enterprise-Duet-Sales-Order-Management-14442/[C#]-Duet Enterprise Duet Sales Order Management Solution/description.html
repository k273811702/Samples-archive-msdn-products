This sample is a functional solution for managing sales orders with Duet Enterprise for Microsoft SharePoint and SAP. It accompanies the set of topics that begins with Walkthrough: Developing a Solution with Duet Enterprise [http://msdn.microsoft.com/en-us/library/hh404062.aspx]. Read the walkthrough for detailed instructions.

Duet Enterprise for Microsoft SharePoint and SAP is both a product in itself, jointly developed by SAP and Microsoft to facilitate interoperability between SAP back-end systems and Microsoft SharePoint Server 2010, and a development platform that allows developers to create customized solutions to take advantage of that interoperability.

A complete Duet Enterprise solution involves development in the SAP environment, using SAP NetWeaver and ABAP Workbench, and development in the context of SharePoint Server 2010, using SharePoint Designer or Visual Studio 2010. This sample demonstrates how to copy announcements from individual sites in a site collection to an Announcements Web Part on the root site.

Building the Sample

Before attempting to build the sample, read Setting Up a Development Environment for Working with Duet Enterprise
[http://msdn.microsoft.com/en-us/library/hh404057.aspx].

To build the sample by using Microsoft Visual Studio

  1. Open Windows Explorer and navigate to your download directory.
  2. Double-click the icon for the .sln (solution) file to open the file in Visual Studio.
  3. In the Build menu, select Build Solution. The application will be built in the default \Debug or \Release directory.
