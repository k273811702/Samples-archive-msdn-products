Introduction

Each code sample consists of approximately 5 to 50 lines of code demonstrating a distinct feature or feature set, in either VBA or both VB and C# (created in Visual Studio 2010). Each sample includes comments describing the sample, and setup code so that you can run the code with expected results or the comments will explain how to set up the environment so that the sample code runs.)

Microsoft® Office 2010 gives you the tools needed to create powerful applications. The Microsoft Visual Basic for Applications (VBA) code samples can assist you in creating your own applications that perform specific functions or as a starting point to create more complex solutions.

Microsoft Excel Samples

  • Excel.DataBars. Given an Excel workbook, work with data bars to include fill, axis position, and the color of negative and positive values.

  • Excel.NewApplicationEvents. Given an Excel workbook, demonstrates the new WorkbookNewChart and WorkbookAfterSave events.

  • Excel.TextFrameProperties. Given a text frame in an Excel workbook, demonstrate the properties available for text that doesn’t fit the frame.
  • Excel.SeriesProperties. In an Excel workbook, demonstrate various properties of a chart’s Series class.
  • Excel.TestAddIn.IsOpen. In an Excel workbook, programmatically determine if an add-in is open.

  • Excel.AddUnique. Given a range of random numbers in an Excel workbook, demonstrate the AddUnique method.
  • Excel.RemoveDuplicates. Demonstrate using the RemoveDuplicates method that enables you to remove duplicate rows from a range of data.
  • Excel.ExportAsFixedFormat. In an Excel workbook, demonstrate the ExportAsFixedFormat method which can export a range to PDF or XPS format.
  • Excel.RemoveDocumentInformation. From an Excel workbook, remove comments, defined name comments, personal information, and document properties.
  • Excel.WorksheetSort. Given a range of data, programmatically sort it is ascending and descending order.
  • Excel.WorksheetFunctionDates. Work with the WorksheetFunction object, focusing on several date functions.
  • Excel.CustomView. Work with the CustomView and WorksheetView classes to create and manipulate custom views in a workbook.
  • Excel.ApplicationProperties. In an Excel workbook, manipulate various application properties that apply to the user interface.
  • Excel.PageSetup. Given a range of numbers in an Excel workbook, demonstrate using various properties of the PageSetup object.
  • Excel.Gradient. In an Excel workbook, work with the gradient fill features of the FillFormat object
  • Excel.ListObjectDisplay. In an Excel workbook containing a table, work with the ListObject object to modify the display properties of the table.
  • Excel.ListObjectSortFilter. In an Excel workbook containing a table, work with ListObject object to programmatically sort and filter the table.
  • Excel.ListObjectTableStyles. In an Excel workbook, work with ListObject object properties such as TableStyles, TableStyle, TableStyleElements, and TableStyleElement.
  • Excel.SparkLines. In a range in an Excel workbook, work with various properties of a sparkline group.
  • Excel.PagesAndPage. Work with the header and footer properties of the Pages and Page objects.
  • Excel.ListColumn. In an Excel workbook containing a table, work with the ListColumn object’s DataBodyRange and Total properties.
  • Excel.WindowProperties. Demonstrate the ActiveSheetView, DisplayRuler, DisplayWhiteSpace properties of the Window object.
