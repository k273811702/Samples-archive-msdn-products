Introduction

The sample code and the accompanying video and MSDN article show how to add Microsoft Unified Communications Managed API (UCMA) 3.0 code to a Windows Workflow Foundation 4 application.

Description

The technical article is divided into four parts about how to call Microsoft Unified Communications Managed API (UCMA) 3.0 code in a Windows Workflow Foundation 4 application.

  • Running UCMA Code in a Windows Workflow Application: Introduction (Part 1 of 4)

    • This article describes how to call Microsoft Unified Communications Managed API (UCMA) 3.0 code in a Windows Workflow Foundation 4 application.

  • Running UCMA Code in a Windows Workflow Application: Creating the Workflow (Part 2 of 4)

    • This section describes how to create a simple Windows Workflow application.

  • Running UCMA Code in a Windows Workflow Application: Adding UCMA Components (Part 3 of 4)

    • Part 3 describes how to add a UCMA project to a Windows Workflow application.

  • Running UCMA Code in a Windows Workflow Application: Code Listing (Part 4 of 4)

    • This is the last in a series of four articles about how to add a UCMA component to a workflow.
