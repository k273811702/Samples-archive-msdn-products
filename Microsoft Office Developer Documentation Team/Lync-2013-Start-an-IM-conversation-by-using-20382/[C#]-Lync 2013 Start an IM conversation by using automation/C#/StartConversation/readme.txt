Conversation Sample "Hello World" Sample

Uses the Automation class in the Lync Model API to start a new conversation with one Lync contact.

Sample location:
64Bit Operating System: %PROGRAMFILES(X86)%\Microsoft Lync\SDK\Samples\Micro\StartConversation
32Bit Operating System: %PROGRAMFILES%\Microsoft Lync\SDK\Samples\Micro\StartConversation

Features
- Uses the Automation API in Lync to start a new conversation with new Lync contacts.
- The conversation can be started with IM modalities.

Warnings:
- The sample runs in Visual Studio 2010.

Prerequisites (for compiling and running in Visual Studio)
- .Net Framework 4.0 or above.
- Visual Studio 2010 or above
- Microsoft Lync 2013 SDK

Prerequisites (for running installed sample on client machines)
- Microsoft Lync 2013 must be installed and running.

Running the sample
------------------
1. Open StartConversation.csproj
2. Hit F5 in Visual Studio
3. Provide sip uri or contact address in the participant box 
4. click start conversation
