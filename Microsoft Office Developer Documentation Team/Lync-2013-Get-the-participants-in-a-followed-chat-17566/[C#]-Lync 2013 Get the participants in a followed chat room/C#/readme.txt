Persistent Chat: Chat room participant roster

This application demonstrates the ability to display a list the participants in a selected chat room.

Sample location:
64Bit Operating System: %PROGRAMFILES(X86)%\Microsoft Lync\SDK\Samples\PersistentChat_GetParticipants
32Bit Operating System: %PROGRAMFILES%\Microsoft Lync\SDK\Samples\PersistentChat_GetParticipants

Features
- Get chat room participants.

Warnings:
- The sample runs in Visual Studio 2008 but the project file included in it is for Visual Studio 2010.
- Copy the PersistentChat_GetParticipants folder to a user folder, outside of Program Files.
- Both Lync and the PersistentChat_GetParticipants project must run with the same priviliges.

Prerequisites (for compiling and running in Visual Studio)
- .Net Framework 3.5 or above.
- Visual Studio 2008 or 2010
- Microsoft Lync 15 Technical Preview SDK

Prerequisites (for running installed sample on client machines)
- Microsoft Lync 15 Technical preview must be installed and running.

Running the sample
------------------
1. Open PersistentChat_GetParticipants.csproj file.
2. Hit F5
3. Select a followed chat room by incrementing or decrementing the Followed Room numeric up/down control.
