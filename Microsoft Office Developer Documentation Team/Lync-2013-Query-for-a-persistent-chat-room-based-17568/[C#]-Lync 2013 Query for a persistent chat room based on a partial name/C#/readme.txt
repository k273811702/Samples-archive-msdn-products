Persistent Chat: Query for a chat room by name.

This application demonstrates the ability to query for a chat room by name.

Sample location:
64Bit Operating System: %PROGRAMFILES(X86)%\Microsoft Lync\SDK\Samples\PersistentChat_RoomQuery
32Bit Operating System: %PROGRAMFILES%\Microsoft Lync\SDK\Samples\PersistentChat_RoomQuery

Features
- Query a Persistent Chat Server for chat rooms by name or partial name.

Warnings:
- The sample runs in Visual Studio 2008 but the project file included in it is for Visual Studio 2010.
- Copy the PersistentChat_RoomQuery folder to a user folder, outside of Program Files.
- Both Lync and the PersistentChat_RoomQuery project must run with the same priviliges.

Prerequisites (for compiling and running in Visual Studio)
- .Net Framework 3.5 or above.
- Visual Studio 2008 or 2010
- Microsoft Lync 15 Technical Preview SDK

Prerequisites (for running installed sample on client machines)
- Microsoft Lync 15 Technical preview must be installed and running.

Running the sample
------------------
1. Open PersistentChat_RoomQuery.csproj file.
2. Hit F5
3. Type a partial or complete chat room name in the text entry field.
4. Press the Go button.
