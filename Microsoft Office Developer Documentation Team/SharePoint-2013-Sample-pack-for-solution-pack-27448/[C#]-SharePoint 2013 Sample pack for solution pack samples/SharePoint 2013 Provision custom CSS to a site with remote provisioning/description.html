This sample demonstrates how to use CSOM to provision custom CSS to a SharePoint site.

Description of the sample

This app for SharePoint sample sets the custom CSS to the host site using a custom user action and an injection pattern. Use it to learn how to deploy a custom CSS file to the host web, embed a custom block to be executed on the host web, and apply custom CSS to the site using the embedded .

Deviations from good practices

A portion of this sample uses a injection pattern. It applies specifically to this scenario.

Prerequisites

This sample requires the following:

  • Visual Studio 2012 or Visual Studio 2013

  • Microsoft Office Developer Tools for Visual Studio

  • A SharePoint 2013 development environment
