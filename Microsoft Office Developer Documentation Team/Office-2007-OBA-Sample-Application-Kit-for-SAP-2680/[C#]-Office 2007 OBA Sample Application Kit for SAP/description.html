The OBA Sample Application Kit for SAP is a version 2.0 of a kit designed to help developers integrate SAP with the 2007 Office system, specifically Excel 2007, PowerPoint 2007, and MOSS 2007. The version 2.0 rev of this kit uses Visual Studio 2008 (specifically Visual Studio Tools for Office (VSTO) 3.0 technology) and adds MOSS integration to the kit.

The version 1.0 of the kit (called the OBA Starter Kit for SAP) can be found on MSDN at the following location:

http://msdn.microsoft.com/en-us/library/bb498189.aspx

The version 1.0 kit uses Visual Studio 2005/VSTO 2005 SE technology and did not include the MOSS 2007 integration with the Business Data Catalog.
 
In addition to the sample code, this download contains the following files:
  • OBA SAK SAP Installation Guide
  • OBA SAK SAP Technical Walkthrough
  • OBA SAK SAP Client Install Bits
