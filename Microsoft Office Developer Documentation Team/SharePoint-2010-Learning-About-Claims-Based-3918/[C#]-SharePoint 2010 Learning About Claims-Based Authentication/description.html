Introduction

This sample and the accompanying article, Learning About Claims-Based Authentication in SharePoint 2010, provide guidance for frequently asked questions about claims-based authentication in Microsoft SharePoint 2010. The accompanying article also provides tips and guidance to help solve problems related to using and configuring claims, and points to other resources for more information.

Description

The accompanying article provides guidance about the following SharePoint 2010 claims tasks:

  • Creating multiple claims authentication web applications in a single SharePoint 2010 farm.
  • Migrating a web application from Windows classic to Windows claims in SharePoint 2010.
  • Using audiences with claims authentication sites in SharePoint 2010.
  • Creating an identity role and a role claim for a SharePoint 2010 claims authentication application.
  • Determining what claims you have in SharePoint 2010. This code sample accompanies the section Tip 5: Determining What Claims You Have in SharePoint 2010 in the associated article.

Requirements

To use this sample, you must have the following:

  • Microsoft SharePoint Foundation 2010
  • Microsoft SharePoint Server 2010
  • Knowledge of C#
