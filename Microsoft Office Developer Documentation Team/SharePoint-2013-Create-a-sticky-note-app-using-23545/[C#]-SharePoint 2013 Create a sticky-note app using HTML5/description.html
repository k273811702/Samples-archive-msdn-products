SharePoint 2013: Create a sticky-note app using HTML5

Demonstrates the use of the HTML5 canvas and a custom list to create a simple "sticky-note" app.

Provided by: Scot Hillier, Critical Path Training

This sample app lets a user add, update, edit, and delete yellow "sticky notes."

Prerequisites

This sample requires the following:

  • A SharePoint 2013 development environment

  • Visual Studio 2012 and Office Developer Tools for Visual Studio 2012

Key components

The sample consists of a SharePoint-hosted app that uses HTML5 capabilities to draw the notes, whose data is saved in a custom list. The libraries wingtip.stickynote.js and wingtip.viewmodel.js contain the code used to perform create, read, update, and delete (CRUD) operations against the list and draw the note graphics.

Configure the sample

This sample does not require any configuration.

Build, run, and test the sample

  1. Open the StickyNotes.sln solution in Visual Studio 2012.

  2. Edit the deployment URL to point to a site in your development environment.

  3. Press F5.

  4. When the app appears, you will see three pre-loaded notes.

  5. Use the New button to add a note.

  6. Select a note by clicking it.

  7. Type into the selected note.

  8. Use the Delete button to delete the selected note.

Troubleshooting

The browser must support HTML5. (For compatibility information, see HTML5 And CSS3 Browser Compatibility Chart.)
