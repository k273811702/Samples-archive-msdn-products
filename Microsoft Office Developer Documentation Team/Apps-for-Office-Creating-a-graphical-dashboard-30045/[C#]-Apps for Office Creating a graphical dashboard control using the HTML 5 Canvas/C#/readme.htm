Description of the sample

This sample shows how to create an interactive graphical control that will display values from an Excel spreadsheet. The control can also be used to change the value using a mouse or touch. Such controls can be used to make a live dashboard that displays business status.

Prerequisites

This sample requires the following:

  • A Windows 8 development workstation.

  • Office 365.

  • Visual Studio 2013.

  • Microsoft Office Developer Tools for Visual Studio 2013 - May 2014 Update

Key components of the sample

The sample zip file contains the following:

  • A Visual Studio 2012 solution file that contains the AppManifest.xml file and source for the app named InteractiveControl.

  • A copy of this readme.

Configure the sample

Follow these steps to configure the sample.

  1. On a Windows 8 workstation, open Visual Studio 2013.

  2. On the File menu, click Open, Project/Solution and then navigate to the folder where you unzipped the project. Select the InteractiveControl.sln file and click Open.

Run and test the sample

Run the solution in debug mode to explore the code sample. Since the app is web based there is no need to build it.
