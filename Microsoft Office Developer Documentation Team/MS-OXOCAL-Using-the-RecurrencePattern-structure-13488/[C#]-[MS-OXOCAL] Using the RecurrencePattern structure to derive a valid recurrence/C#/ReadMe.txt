Calcuating FirstDateTime: Derives the FirstDateTime field for a daily, weekly, or monthly/yearly recurrence.
This sample is available in the following language implementations:
C#

Files:

Calculating FirstDateTime.cs		// Contains the sample code
Calculating FirstDateTime.csproj	// Visual Studio Project File
Calculating FirstDateTime.sln		// Visual Studio Solution File

Open the *.sln file in Visual Studio. In the Properties pane of Visual Studio, change the Site URL value to absolute address of your development test site. For example, "http://MyDevServer/". Be sure to include the closing forward slash.

To build the sample using Visual Studio 2008 (preferred method):
================================================
     1. Open Windows Explorer and navigate to the  directory.
     2. Double-click the icon for the .sln (solution) file to open the file in Visual Studio.
     3. In the Build menu, select Build Solution. The application will be built in the default \Debug or \Release directory.
