## Description

<h2>Introduction</h2>
<p>This is a sample Visual Studio 2010 classifier for diff/patch files. It provides formatting for the various line types found in diff files using the new classification API found in VS2010.
<br>
<br>
The binaries for this sample can also be installed via the extension manager.<br>
<br>
</p>
